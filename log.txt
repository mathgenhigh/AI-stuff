- Problem description (human language)

    The goal of the fuzzy controller is to determine the fan speed in a room based on the
    current temperature and humidity level.

    A human does not control the fan using exact formulas, but rather using vague rules 
    such as: “If the temperature is high and the humidity is high, the fan should run fast.”

    Fuzzy logic allows us to model this human-like reasoning using linguistic variables and 
    rules instead of precise mathematical equations.

    - Input and output variables

        - Inputs:

            1. Temperature (°C)
                - Range: 0 – 40

            2. Humidity (%)
                - Range: 0 – 100

        - Output:

            Fan speed
                - Range: 0 – 100 (%)

- Fuzzy sets & membership functions

    I will use simple triangular membership functions.

    - Input 1: Temperature

        Linguistic value        Range        Description

        Low                     0-15        Cold room        

        Medium                  15-30        Comfortable

        High                    30-40        Hot room 


        - Membership functions:
            - Low: triangle (0, 0, 15)
            - Medium: triangle (15, 25, 30)
            - High: triangle (30, 40, 40)


    - Input 2: Humidity


        Linguistic value        Range       Description

        Slow                    0-30        Low speed

        Medium                  30-70       Normal speed 

        Fast                    70-100      High speed

        - Membership functions:
            - Low: triangle (0, 0, 30)
            - Medium: triangle (30, 50, 70)
            - High: triangle (70, 100, 100)

    - Output: Fan speed

        Linguistic value        Range       Description

        Slow                    0-30        Low speed

        Medium                  30-70       Normal speed 

        Fast                    70-100      High speed

        - Membership functions:
            - Low: triangle (0, 0, 30)
            - Medium: triangle (30, 50, 70)
            - High: triangle (70, 100, 100)

- Fuzzy rules

    These rules reflect human reasoning:


    Rule       IF Temperature       AND Humidity        THEN Fan Speed 

    R1         Low                  Low                 Slow 

    R2         Low                  High                Medium 

    R3         Medium               Low                 Medium 

    R4         Medium               Medium              Medium

    R5         High                 Medium              Fast 

    R6         High                 High                Fast
